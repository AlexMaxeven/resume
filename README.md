# 💼 Резюме-сайт

Сучасний інтерактивний веб-сайт-резюме з анімаціями та захистом від XSS/CSRF атак.

## 🎯 Опис

Персональний сайт-резюме, що демонструє професійний досвід у Frontend розробці, MikroTik мережах, кібербезпеці та психології. Включає інтерактивні компоненти, WebGL анімації та практичну демонстрацію захисту від веб-атак.

## ✨ Особливості

- **Інтерактивні анімації** - MagicBento картки, Particles, Galaxy, LiquidEther
- **WebGL фони** - 3D частинки, зоряне небо, рідинні ефекти
- **Захист від атак** - XSS/CSRF з практичною демонстрацією
- **Багатомовність** - Українська та Англійська мови
- **Адаптивність** - Повна підтримка мобільних пристроїв (320px+)

## 🛠️ Технології

- **React 19** + **Vite** + **React Router DOM**
- **GSAP** + **Framer Motion** - анімації
- **Three.js** + **OGL** - WebGL графіка
- **DOMPurify** - захист від XSS
- **CSS Modules** - модульні стилі

## 📄 Сторінки

- **Home** - Резюме, навички, проєкти (Particles фон)
- **About** - Досвід роботи, технічні навички (Particles фон)
- **Contact** - Освіта, розвиток, мови, контакти (LiquidEther фон)
- **Kali** - Кібербезпека, інструменти, практичні навички + **демонстрація захисту XSS/CSRF** (LiquidEther фон)
- **Psychology** - Гештальт-терапія, досвід, напрямки роботи (LiquidEther фон)

## 🔒 Безпека

Проект включає систему захисту від XSS та CSRF:
- **DOMPurify** - автоматична санітизація HTML
- **CSRF токени** - захист від міжсайтових запитів
- **Безпечний рендеринг** - компонент SafeHTML

Демонстрація захисту доступна на сторінці **Kali** (внизу сторінки).

Детальна документація: [SECURITY.md](./SECURITY.md)
